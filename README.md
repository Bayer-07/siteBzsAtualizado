# Novo site BZS Tecnologia

Site institucional em React + Vite para a BZS Tecnologia.

## Como rodar

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev`: inicia o servidor local do Vite.
- `npm run build`: gera a versão de produção em `dist/`.
- `npm run preview`: visualiza o build localmente.

## Páginas

- `/`: página inicial.
- `/sistemas/bzs-recicle`: BZS Recicle.
- `/sistemas/selfie`: Selfie.
- `/sistemas/controle-agua-gas`: Controle Água e Gás.
- `/sistemas/transporte-escolar`: Gestão de Transporte Escolar.
- `/sistemas/bibliotecas`: Gestão de Bibliotecas.

## Estrutura

- `src/App.jsx`: componentes e conteúdo da landing page.
- `src/data/systems.js`: base de conteúdo das páginas dos sistemas.
- `src/styles.css`: layout responsivo e identidade visual.
- `src/assets/`: logos reaproveitados do site atual e imagem nova do hero.
- `public/favicon.png`: ícone do site.
